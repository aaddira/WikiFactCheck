📋 How to Fact-Check This Citation

  1. Carefully read the quoted Wikipedia passage and compare it to the source content
  2. Select a label: Supported, Not Supported, Mixed, Unverifiable, or Source Link Error
  3. Quote exact verbatim text from the source that supports or contradicts the passage
  4. Explain how the quote relates to the passage, note any uncertainties
  5. Skip any statement you're unsure about or don't wish to evaluate
Fact-Check Decision
📖 View Label Definitions
✅ Supported
All highlighted claims are correctly grounded in the source text.
❌ Not Supported
All highlighted claims are missing from the source or contradicted by it.
⚠️ Mixed
A claim or more in the response are correctly grounded in the source text, but other claims (either explicit or implicit) are not.
🔍 Unverifiable
The response has no factual claims to verify due to consisting of opinions, subjective claims, unquantifiable statements, responding to an unanswerable question, or similar.
Example: "Lord of the Rings is the best book ever written"
🔗 Source Link Error
The source document is either a broken link or clearly irrelevant content that was mistakenly cited.
Example: A web page about hamsters is linked in a claim for the red pandas article, no red pandas are mentioned in the source!
